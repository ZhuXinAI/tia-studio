# Scope v3 to the embedded Pi SDK

The v3 harness ships complete local conversations through Pi's embedded SDK: session create/resume, text and image input, streaming text/reasoning/tools, steering and follow-up queues, cancellation, model/thinking selection, extension interactions, restart resumption, and live thread switching. Branch-tree, manual compaction, and session-stat UI may be considered separately; AI SDK HarnessAgent, process RPC, and remote execution are excluded from the architecture entirely and must not leave adapters, feature flags, dependencies, or roadmap placeholders.
