# v0.1.22

## Highlights

- Redesigned the chat sidebar into a master-detail flow so `/chat` acts as an assistant directory and each assistant opens into its own thread-focused detail view.
- Added searchable, virtualized assistant thread lists plus a quick assistant switcher in detail mode for faster navigation across busy chat histories.

## Fixes

- Restored channel tool progress updates so channel users see tool inputs, outputs, and localized error details again during assisted runs.
- Updated the Claws create-flow coverage to match the current four-step onboarding flow, keeping release CI green.

## Release Notes

- Updated the release workflow to publish a single macOS universal artifact, which helps keep desktop auto-update metadata in sync across platforms.
