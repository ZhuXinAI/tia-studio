# TIA Studio 0.1.17

This release improves channel behavior and documentation for the current four supported channels.

## Highlights

- Added configurable `groupRequireMention` support for Lark, WhatsApp, and Wecom group chats
- Kept Telegram intentionally DM-only for now while group behavior is still under review
- Added channel support documentation in `README.md`, `README_zh.md`, and the new `CHANNEL.md`
- Added UI controls for group mention gating on supported channels

## Channel behavior

| Channel | Direct Chat | Group Chat | Default Group Trigger |
| ------- | ----------- | ---------- | --------------------- |
| Lark | ✅ | ✅ | Bot must be `@` mentioned |
| Telegram | ✅ | 🚫 | Group support disabled for now |
| WhatsApp | ✅ | ✅ | Bot must be `@` mentioned |
| Wecom | ✅ | ✅ | Bot must be `@` mentioned |

## Notes

- Telegram channel configuration still exists in the same channels UI, but group-specific gating is hidden because Telegram group runtime support is currently disabled.
- For more channel details, see `CHANNEL.md`.
