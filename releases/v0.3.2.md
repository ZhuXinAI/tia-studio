# v0.3.2

## Highlights

- Added the new Wechat channel integration with QR-based authentication, long polling, and channel auth state handling.
- Improved the assistant setup flow so Wechat setup can be opened directly from the assistant modal and newly attached Wechat channels open setup immediately.
- Refreshed Wechat connection status in the UI as soon as QR binding completes and added upstream-style typing signals to reduce reply latency.

## Notes

- `v0.3.2` is a patch release focused on shipping and stabilizing the new Wechat channel experience.
