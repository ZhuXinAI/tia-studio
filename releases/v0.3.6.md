# v0.3.6

## Highlights

- Chat and workspace empty states now open an actual draft conversation at `/chat/new` or the workspace equivalent. A thread is created only after you send the first message.
- Moved model and permission controls beside Send, constrained long model lists to a scrollable menu, and make Full Access visibly distinct.
- Hid the unavailable voice-input control and ACP providers while their supporting extensions are not available.
- Localized the MCP, Channels, and appearance-token surfaces, removed the redundant MCP title, and consistently use TIA Studio in user-facing copy.

## Notes

- This release also adds clearer separation between Settings navigation groups.
