# v0.2.0

## Highlights

- Added a new Group workspace where multiple assistants can collaborate in a shared room with dedicated groups, threads, member selection, max auto-turns, and live room status updates.
- Added group mentions and a new group runtime so assistants can hand work to specific teammates while keeping one shared transcript in sync.
- Added Wechat-KF channel support in the channel management flow, including relay server credentials, inbound event handling, and outbound replies.

## Platform And Release Improvements

- Upgraded the CI and release workflows to the newer GitHub Actions runtime, enabled macOS entitlements and notarization checks, and turned on notarized macOS releases by default.
- Added a separate tagged Intel macOS asset upload flow so x64 Mac builds can be attached to the same release without changing the standard release path.

## Notes

- `v0.2.0` marks the first minor release for the new multi-assistant group workflow and expanded channel support shipped after `v0.1.29`.
