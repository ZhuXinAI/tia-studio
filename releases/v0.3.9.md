# v0.3.9

## Highlights

- Added managed MCP connection updates, including OAuth-backed remote connection handling and clearer runtime health feedback.
- Replaced the static Skills marketplace with a live `skills.sh` catalog cached for 24 hours. The catalog retains the first 200 ranked skills, shows the top 20 by default, and searches the complete cached page.
- Polished the application shell, thread interactions, Schedules, and provider settings surfaces.

## Notes

- Skills installation and removal continue to use TIA-owned global storage and require the existing confirmation flow.
