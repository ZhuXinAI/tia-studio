# v0.3.4

## Highlights

- Rebuilt the desktop agent runtime around the embedded Pi SDK, with app-owned provider configuration, authenticated HTTP/SSE transport, and explicit Standard or Full Access permissions.
- Added first-class Schedules, including persisted scheduled-thread provenance, clearer schedule management, and a clock indicator on scheduled conversations.
- Expanded workspace and skills workflows with a refreshed skills marketplace, improved workspace navigation, and updated creation controls.
- Simplified and polished the chat experience with the official assistant-ui thread runtime, clearer tool and reasoning traces, follow-up suggestions, and more reliable thread naming.
- Improved cross-platform release reliability, including timezone-safe schedule coverage and bounded cleanup retries for locked Windows test databases.

## Notes

- `v0.3.4` is a substantial desktop runtime and workflow release that completes the Pi SDK cutover and ships the new scheduling experience across macOS and Windows.
