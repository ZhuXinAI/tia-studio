# v0.1.20

## Highlights
- Added Codex-powered coding support for assistants, including persisted coding settings, Codex CLI availability checks, and runtime wiring for a dedicated coding subagent.
- Added OpenRouter as a supported provider across built-in defaults, provider configuration, model resolution, and claw/provider selection flows.
- Improved desktop auto-update UX with live update-state syncing, a downloaded-update state, and restart-to-update support.

## Chat And Channel Improvements
- Added localized channel tool progress and error updates, plus smarter interruption handling for in-flight channel conversations.
- Returned thread-to-channel binding metadata from the API and surfaced remote-thread badges in the thread UI.
- Virtualized long thread message lists to keep chat history rendering and thread switching more responsive.

## Fixes
- Fixed assistant updates when workspace, coding, and MCP object config payloads are submitted together.
- Hardened validation, persistence, and test coverage around coding assistants, providers, updater flows, and thread chat behavior.
