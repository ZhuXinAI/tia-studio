## Image Guidelines

- **Size**: 256x256px or 512x512px (square aspect ratio)
- **Format**: PNG preferred for transparency support
- **Background**: Transparent background is ideal for better UI integration
- **Quality**: High resolution for crisp display on retina screens
- **Fallback**: If an image is missing, the UI will show initials as fallback

## Usage

These images are displayed as avatars in the provider list at:
- Settings > Providers page
- Each provider shows its logo in a circular avatar (40x40px display size)
